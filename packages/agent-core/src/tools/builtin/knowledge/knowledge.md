Search the project's knowledge bundle — the OKF concepts under `sdd/` (decisions, tasks, the active proposal).

Use this to recall what was already decided, which tasks exist and their status, and why — instead of re-reading files one by one or re-asking the user. Pass an optional `type` (`Decision`, `Task`, `Proposal`) to scope the search to one kind of concept.

The bundle's concept index is also summarized in your system context. Use this tool to locate the specific concept, then read its file for the full detail.
